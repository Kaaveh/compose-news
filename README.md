# Compose News
This is a little app about news that implemented with Jetpack Compose.

Using:
- Jetpack Compose
- CLEAN architecture
- MVVM architectural pattern
- Coroutine Flow
- Room database
- Hilt Dagger
- Navigation
- Retrofit
